Component code

import {{ '{' }}NgxChartData{{ '}' }} from '@vanavara/ngx-polar-chart';

@Component({{ '{' }}
...
{{ '}' }})
export class MyComponent {{ '{' }}
   public groupDateSample: NgxChartData = null!;
   public sampleGroupAItems: NgxGroups<Date>[] = [];
   public sampleGroupBItems: NgxGroups<Date>[] = [];

   public customSettings: NgxPolarChartSettings = null!;
{{ '}' }}

Library types

type NgxChartData = NgxDateGroup | NgxSeriesGroup;
As we work with date NgxChartData is a type of NgxDateGroup
this.sampleGroupAItems: NgxGroups<Date>[]
this.sampleGroupBItems: NgxGroups<Date>[]

In this example there are two groups with items. Each item in the group describes a day of current month and a value which that item has. A total count of items in each group is equal to the days number in current month.

for (let i = startDate; i <= endDate; i++) {{ '{' }}
      sampleGroupAItems.push({{ '{' }}
         key: new Date(moment(now).year(), moment(now).month(), i),
         value: Math.floor(Math.random() * 101),
      {{ '}' }});
   {{ '}' }}
      sampleGroupBItems.push({{ '{' }}
         key: new Date(moment(now).year(), moment(now).month(), i),
         value: Math.floor(Math.random() * 101),
      {{ '}' }});
{{ '}' }}

this.groupDateSample = {{ '{' }}
   groups: [
      {{ '{' }}
         name: 'Sample group A',
         items: sampleGroupAItems,
      {{ '}' }},
      {{ '{' }}
         name: 'Sample group B',
         items: sampleGroupBItems,
    {{ '}' }}]
{{ '}' }}

NgxPolarChartSettings code

In this example we don't want to highlight average, maximum and minimum values on the chart

public customSettings: NgxPolarChartSettings = {{ '{' }}
   barsSettings: {{ '{' }}
      average: {{ '{' }}
      showColorBar: false,
      {{ '}' }},
    maximum: {{ '{' }}
      showColorBar: false,
      {{ '}' }},
      minimum: {{ '{' }}
      showColorBar: false,
      {{ '}' }},
      weekend: {{ '{' }}
      showColorBar: false,
      {{ '}' }},
   {{ '}' }},
{{ '}' }};

We have more than one group

Just create a new group with items like sampleGroupAItems and add into groupDateSample variable
