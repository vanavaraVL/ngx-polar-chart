Template code

<ngx-polar-chart-stack
   [chartData]="stackDateSample"
   [chartSettings]="customSettings">
<ngx-polar-chart-stack>

Component code

import {{ '{' }}NgxChartData{{ '}' }} from '@vanavara/ngx-polar-chart';

@Component({{ '{' }}
...
{{ '}' }})
export class MyComponent {{ '{' }}
   public stackDateSample: NgxChartData = null!;
   public sampleGroupAItems: NgxGroups<Date>[] = [];
   public sampleGroupBItems: NgxGroups<Date>[] = [];

   public customSettings: NgxPolarChartSettings = null!;
{{ '}' }}

Library types

type NgxChartData = NgxDateGroup | NgxSeriesGroup;
As we work with date NgxChartData is a type of NgxDateGroup
this.sampleGroupAItems: NgxGroups<Date>[]
this.sampleGroupBItems: NgxGroups<Date>[]

In this example there are two groups with items. Each item in the group describes a month of particular year/day and a value which that item has. A total count of items in each group is equal to 12 months. And we want to display these two groups by year on our chart.

for (let i = startMonth; i < endMonth; i++) {{ '{' }}
      sampleGroupAItems.push({{ '{' }}
         key: new Date(moment(now).year(), i, 1),
         value: Math.floor(Math.random() * 101),
      {{ '}' }});
   {{ '}' }}
      sampleGroupBItems.push({{ '{' }}
         key: new Date(moment(now).year(), i, 1),
         value: Math.floor(Math.random() * 101),
      {{ '}' }});
{{ '}' }}

this.stackDateSample = {{ '{' }}
   groups: [
      groupBy: NgxGroupBy.year,
      {{ '{' }}
         name: 'Sample group A',
         items: sampleGroupAItems,
      {{ '}' }},
      {{ '{' }}
         name: 'Sample group B',
         items: sampleGroupBItems,
    {{ '}' }}]
{{ '}' }}

NgxPolarChartSettings code

Stack doesn't highlight average, minimum and weekends (as it's logically). So default settings could be pretty much good.

We have more than one group

Just create a new group with items like sampleGroupAItems and add into stackDateSample variable
